# monogatari-ppt — 物语风格演出规范

把《物语系列》的视觉与文本语言变成**可执行、可校验的硬约束**。
一切规则来自实测证据(逐帧逆向 + 字库实测),细节见 references/;
本文件是规则层,生成时违反任何一条"铁律"即返工。

## 何时用 / 何时不用

- 用:演出型课件(字卡驱动、讲述者控节奏)、章节卡/标题卡、概念短片分镜
- 不用:信息密度高的汇报 PPT(走 milwaukee-ppt)、需要图文混排的页面(本风格字卡禁图文混排)

## 六条铁律(违反即返工)

1. **只有硬切**。禁止 fade/slide/zoom 过渡、禁止 CSS transition;切换 0ms。
   (证据:46 切点抽检无混合帧;全片零叠化)
2. **字卡时长与字数无关**。禁止"每字 X 秒/字数÷阅读速度"——实测回归 R²=0.001。
   时长查【镜头类型表】按叙事功能定:吐槽 8–11 帧、叙述 0.8–2.5s、悬念短句反而最长、
   长句故意给读不完的时长。
3. **全程衬线明朝体**,零无衬线。冲击卡也不例外(基线测试最易在"冲击字"上破戒)。
4. **上下黑色遮幅恒定**(合计约画面高 10%),无一帧例外。禁止全出血。
5. **一卡一纯色底**,低饱和:红是 `#a00000` 不是 `#FF0000`,白底是 `#e0e0e0` 不是 `#FFFFFF`。
   禁渐变底、禁噪点暗角"胶片感"(实测不存在)。
6. **横排只居中,竖排只右起**。禁左/右对齐横排;竖排多列列首阶梯下移。

## 镜头类型速查(时长按此表,不按字数)

| 类型 | 用途 | 时长 |
|---|---|---|
| 独白字卡 | 叙述/铺陈 | 0.8–2.5s,中位 1.4s |
| 快闪吐槽卡 | 吐槽藏在快闪里,差点读不到 | 0.27–0.37s(8–11 帧),无角注 |
| 设问卡 | 单字/短词+问号,浅灰底红字 | ~1.8s |
| 对仗判词卡 | 同色同版式成对,中插一镜素材 | 各 0.9–1.2s |
| 人名/标题卡 | 全片最长字卡,带罗马音编号角注 | ~5.3s |
| 素材镜 | 承接上一字卡的名词(说什么切什么) | 0.9–3.4s |
| 连切爆发 | 同主题素材 4–5 连刀 | 每刀 5–6 帧 |
| 黑闪帧 | 段落切分 | 1 帧 |
| 抖动闪帧 | 揭示前同镜 2–3 帧错位连闪 | 2–3 帧 |
| 骤停长镜 | 叙事重音,打断节奏 | >2× 均长(3.4–9.3s) |

节奏模板:开场字/素 1:1 交替 → 中段加速(字卡连发/素材连切)→ 收尾骤停长定格。

## 色彩语义(底色追随台词语义)

| 底色 | 功能 |
|---|---|
| 段落主情绪色(如橙 `#e07010`) | 同一情绪段共享一个底色 |
| 同色系加深(`#b04000`) | 段内情绪升级 |
| 语义转折换色(说到"凉"→蓝 `#2070a0`) | 色彩转折点=语义转折点 |
| 红 `#a00000` 米白字 | 指向核心人物/判词对仗 |
| 黄 `#f0b000` | 吐槽快闪专用 |
| 灰 `#303030`/`#a0a0a0` | 中性叙述/悬念递进 |
| 浅灰 `#e0e0e0` 红字 | 设问反差卡 |
| 黑 | 闪帧/收尾 |

## 角注体系(不要自创格式)

页眉+页脚小字,精确格式含「序號」二字:`開場序號 玖拾玖` / `過場序號 零零肆` /
`終場序號 零零壹`(三段命名:開場→過場→終場);**序号故意不连续**
(玖拾玖→捌拾玖→零零壹…),制造"从长剧集抽帧"的错觉。快闪卡与素材镜不配角注。
人名卡主名下挂小写罗马音+括号编号「( qian. ) ( 01 )」。
**素材镜豁免**:实拍/截图/模拟素材属素材槽位,不受衬线铁律、台词体禁则与角注体系约束,无角注。
频闪关键帧中的"白帧"用米白 `#f0ede6`,不用纯白(铁律 5 同样适用于闪帧)。

## 文本怎么写

**REQUIRED:** 卡面与台词文案规则已独立为 **monogatari-prose** skill(场记板程式/台词体 11 条/
怪异命名法/繁体旧字形与假名规则/罗马字 wapuro 式/长独白体/三声部);写文字前先读它的 SKILL.md。
(references/text-style.md 现为指向该 skill 的指针。)
要点:短句断言不解释、屏上禁感叹号、「不是 X。是 Y。」反转、诊断式收尾「病名,凡。」、
卡面停顿用双全角空格禁顿号。

## 红旗清单(出现即停手返工)

- "这卡 N 个字,所以停 N×0.3 秒" → 查镜头类型表,不是算术题
- "冲击力大的字用黑体" → 全片零无衬线
- "加个淡入更柔和" → 只有硬切
- "纯红更有冲击力" → `#a00000`,实测无纯红
- "角注写 cut.01/scene 1" → 用「序號+大写汉数」体系
- "横排左对齐更有设计感" → 横排只居中
- "文字墙做成 3 帧闪" → 文字墙是定格内容页;闪帧只有纯黑/图形/同镜抖动三种
