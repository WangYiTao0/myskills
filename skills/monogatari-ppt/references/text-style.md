# 物语文本程式(卡面与台词文案规则)

卡面文字、台词、章节文案按本规则生成「物语腔」。视觉与时长规则见 SKILL.md 与
video-style-spec.md,本文只管**文字怎么写**。经考据/日语/中文三镜头对抗评审修订,
关键改动见各节「※」。§五的五怪异表为一次实际应用示例(AI 内训课件),程式本身通用。

## 一、卡面程式(场记板文本)

**黒齣卡(章题卡)**

| 区位 | 程式 | 例 |
|---|---|---|
| 顶部小字 | 「黒 齣」——卡种自指标记(原作 placeholder 帧惯例) | 黒 齣 |
| 中部主字 | 怪异**单字**,占屏 50-70% | 凡 |
| 中部副Ⅰ | ( <romaji>. ) 小写音读+句点 | ( bon. ) |
| 中部副Ⅱ | ( <全篇章号大写数字> ) | ( 壹 ) |
| 底部 | 「<章名二字>　其 ノ <N>」——章名在前、片假名ノ、章内分回数 | 閒 談　其 ノ 壹 |
| 底部副 | chapter N — <english> 全小写 | chapter one — chatbox |

※「凡齣」废止:齣指帧本身,黒齣/赤齣的首字是帧的颜色,不是内容标签;主字回归单字。
※「其ノN」是**章内分回数**(原作语义),每章从其ノ壹重数;章中再插卡则递增其ノ貳——
全篇章序由副Ⅱ承担,不重复。終章底部:「收 束　最 終 話」。

**幕间垫卡**:中部主字「黒 齣」+ 底部「此 齣　　無 畫」。
※原拟「動畫番號 無シ」降级为讲述者备注彩蛋,不上屏(番號在大陆语境有歧义联想)。
※送り仮名固定片假名(無シ),禁平假名形「無し」。

**赤齣卡(强调/判词)**

三段递进,一段比一段短:场景词 → 对比词 → 判词(≤4 字)。
例:同 樣 的 工 具 → 會用 與 不會用 → 差 十 倍

**設定集白卡(章末)**

顶部「<N> 章 ・ 結」(終章「終 章 ・ 結」;「收束」专留終章章名);
主句 = 章核心命题,≤8 字,停顿用**双全角空格**(禁顿号,见 II-5);
右下印章「<怪异>之怪異」竖排细框——※本课件自创识别符,非原作惯例,保留。

## 二、台词体(黑底正文页)

1. **短句**。一句一个意思。能断句就断句。
2. **断言,不解释**。「病不在 AI,在问法。」——解释留给讲述者口播,屏上只留判词。
3. **「不是 X。是 Y。」** 反转结构是主武器;变体「是 X,非 Y」。
4. **省略主语**;诊断式收尾:「病名,凡。」
5. 顿挫用全角空格与「——」;**屏上文本禁感叹号**(讲述者口播不受限);问号保留给自问自答。
6. 自问自答:「为什么?因为你问错了。」
7. 卡面数字汉字化(差十倍/壹貳參),正文可用阿拉伯数字。
8. **报病名带注解拍**(首次):「病名,凡。凡庸的凡。」「病名,猜。瞎猜的猜。」
   「病名,盲。盲点的盲。」「病名,忘。健忘的忘。」「病名,囚。囚禁的囚。」此后单字裸用。
9. **限定否定**(西尾式收小范围):「它不是什么都知道。只知道你说过的。」(參章)
10. **吐槽两行制**(一问装傻一句拆台):「——让它猜,不行吗?」「行。然后你改十遍。」(貳章)
11. **同字回环**(每章至多一处):「猜。猜你要什么,猜你不要什么,猜你猜它能猜对什么。」(貳章)
    咬字梗「不好意思,咬到舌头了」可作全场一次性彩蛋。

## 三、命名程式

- **怪异名**:单字,指症状的病理本质而非现象——凡 / 猜 / 盲 / 忘 / 囚 / 解
- **章名**:二字,指工具的本相——閒談 / 命令 / 文脈 / 技藝 / 手足(終章:收束)
- **病历体**(文字墙):「<怪异>之怪異 ・ 病歴」起头,竖排逐条症状,
  以「病不在 X,在 Y。」收束。(※病歴用日文字形:HGPMinchoB 缺「歷」)
- 口播固定注解:「齣」首现时一句带过(「戏文里一齣戏的齣——每个病,一齣戏」);
  「手足」固定口播「给 AI 装上手和脚,它才出得了对话框」。

## 四、字形与语种

- **汉字一律繁体/旧字形**(此层中日通用,不刻意区分);
  **日语感只由两个载体制造**:①片假名(其ノ壹的ノ、無シ的シ),②罗马字括注。
  不靠挑选「日本汉字」提供日味。
- **卡面用字以 HGPMinchoB 字库实测为准**(JIS 字库,中文常用字多有缺):
  已测可用:閒 齣 會 與 點 終 樣 號 畫 藝 壹貳參 籠 它 們 説 診 歴;
  已测缺字:**你 们 說 诊 歷 设**——上卡前规避或换 JIS 字形(說→説)。
  检查命令:`python3 -c "from fontTools.ttLib import TTFont; cmap=TTFont('assets/HGPMinchoB.woff').getBestCmap(); print([c for c in '<待测文字>' if ord(c) not in cmap])"`
- **罗马字**:ヘボン式,长音一律重写式 ou/uu(mou・bou・shuu),与原作官方习惯(Ougi)一致;
  禁长音符 ō/ū,禁削短 mo/shu。正文罗马字全小写、句点结尾。
- 正文一律简体中文(走 Hiragino 字体栈,不受 JIS 限制)。

## 六、长独白体(旁白/口播层)

卡面是从独白里**抽帧**出来的句子——先写独白,再抽帧成卡,不要反过来拼凑。
独白体九式(样本证据:千石抚子视频旁白「蛇在热死之前…」段):

1. **让步起手**:「话是这么说,实际…」「不,倒不如说…」——接住上一句再拐弯。
2. **类比即推理**:日常类比(婴儿找妈妈)不是修辞,是推理的轨道——把类比当事实
   继续往下推,推到荒谬也不回头。
3. **先升维再落地**:从具体(蛇避暑)一步跳到普遍命题(「任何人和动物在思考前
   都会遵循本能呢」),一句完成,不解释。
4. **假设连锁**:「但这里是 X 的情况。那么,一定是 Y 吧?」——用确信的语气推进
   纯属虚构的推论,一环扣一环。
5. **排比否定堆叠**:「无人告诉…,无人告诉…,不知道…,自然也不会…」三连以上,
   因果一气贯通。
6. **回望条款**:「就算侥幸…,之后回望,也一定会觉得…吧?」——给假设加上时间维度,
   从未来看现在。
7. **对句定理**:推理收口必是一组对仗因果句——「因为无知,所以无罪。/因为不知道
   是错误的,所以没有愧疚感呢。」温柔语气,残酷结论。
8. **反转揭示**:「明明之前是…,现在却摇身一变成了…——X」破折号最后一格才亮出
   真正的主语。
9. **语气配律**:推测句尾「吧」、自答句尾「呢」;全程自言自语的音量;独白层允许
   夸张抱怨处偶发感叹号(「复仇吧!」),卡面层仍禁。

抽帧规则:独白写完后,挑出其中**判词性最强的短句**上卡(对句定理与反转揭示
几乎必上卡),长推理留给口播;卡与卡连读应能还原独白的骨架。

## 七、声部档位(三种口吻,检索样本)

独白体九式是叙述者的默认声部;以下三档按场合切换。引文为网络检索逐字摘录,出处见尾注。

**甲、欺诈师声部(贝木泥舟式诡辩)——讲成本/收益/价值的段落用**

- 样本(恋物语):「因为钱可以代替一切。钱是能成为一切事物的代用品的,万能卡片。
  能买东西,能买命,能买人,能买新,能买幸福,能买梦想——世界上没有什么比它更重要,
  更无可替代了。所以我喜欢它。」(※「能买新」一处转录存疑,待校)
  「在我看来会相信书啊、文章啊、语言啊等东西的人反而不可理喻——说这话的我本人,
  也就是欺诈师贝木泥舟,不过这也不一定是真实的。」
- 式样:①排比堆叠后用小学生式直白收口(「所以我喜欢它」)——宏大铺陈×幼稚结论的反差;
  ②**自我消解尾注**:断言之后立刻补「不过这也不一定是真实的」,自毁可信度反而可信;
  ③价值反转命题(伪物论:「只要蕴藏着想成为真物的意志,伪物就比真物还要来得真实」);
  ④全程无声调起伏,禁感叹。
- 课件用法:「因为它可以代替一切重复劳动。能写,能算,能画,能查——所以我喜欢它。
  不过这话出自一个靠它干活的人之口,也不一定是真实的。」

**乙、斗嘴仪式(八九寺式对话体)——演示人机对话翻车的段落用**

- 程式(化物语固定桥段):念错名字 →「失礼,咬到舌头了」→ 被拆台 → 故意变本加厉再错
  → 再被拆台。五拍仪式,双方都知道是仪式,仍一丝不苟执行。
- 式样:对话喜剧=**同一个错误的仪式化重复**;装傻方道歉永远不诚恳,拆台方吐槽永远短促。
- 课件用法(貳章猜):
  「——帮我写个报告。」「好的,以下是十条通用建议。」
  「不是这种。」「失礼,猜错了。」
  「你根本没问我要什么。」「失礼,咬到舌头了。」
  「你没有舌头。」

**丙、专家声部(忍野咩咩式收束)——終章专用**

- 样本(化物语/伤物语):「人は一人で勝手に助かるだけ。誰かが誰かを助けることなど
  できない。」通行中译:「人只能自己擅自得救,谁也无法拯救别人。」
- 式样:①先全称否定(谁都救不了人)再留唯一活路(自己擅自得救);
  ②关键词「擅自/勝手に」——把得救贬为当事人的任性,从而免除拯救者的居功,温柔的免责;
  ③轻佻语气说严肃结论,断言后可接一句玩笑卸力。
- 終章既定文案「没人能教会你用 AI。你只是自己,擅自学会了。」即此声部应用。

尾注:样本检索自句子控/萌娘百科条目摘要、bilibili BV1N4411L7CW(日文原文对照)、
百度百科;主要站点反爬,引文来自搜索摘要层,人工复核后可升级置信度。

## 八、禁则

- 屏上禁感叹号;禁汇报腔(赋能/抓手/闭环);禁长定语从句
- 屏上不写"接下来我们看…"类舞台调度语(那是讲述者的活)
- 不堆形容词;气势靠字重和留白,不靠副词
- 卡面停顿禁顿号(用双全角空格/「——」)
