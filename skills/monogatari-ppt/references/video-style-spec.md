# 物语风格演出逆向 Style Spec

样本:《用物语风格记录下由天气炎热联想到千石抚子的过程》(B 站二创,852×480,30fps,75.1s,46 切点 @scene 0.15)
方法:场景切分抽帧 47 张 → 逐帧五维分类 → 时间戳×分类合并统计。
所有规则均有帧号/时间戳证据;**禁止凭原作记忆扩写**。

## 3. 排版语法

- **遮幅**:全片上下黑边 letterbox(合计约画面高 10%,480p 下各 ~24px),无一帧例外。
- **字体**:全部衬线明朝体;无任何无衬线字出现。
- **横排**:单行/双行/三行居中,垂直略偏上;字距明显拉宽(0014/0017/0026)。
- **竖排**:右起多列,**列首阶梯式逐列下移**(0002 四列/0004 三列),用于开场长句抱怨;文字占右半幅,左半留白。
- **角注体系**:页眉+页脚各一行小字「開場/過場/終場序號 + 大写汉字数字」;
  序号**故意不连续**(玖拾玖→捌拾玖→柒拾柒→零零壹→零零貳→零零叁→零零玖→捌拾捌→捌拾玖→零零壹),
  营造"从更长的剧集里抽帧"的错觉(证据:0000/0002/0004/0006/0007/0009/0023/0026/0032/0034/0045)。
  快闪吐槽卡(0028/0029)与部分短卡**无角注**——读不到的卡不配序号。
- **人名卡**:主名下挂拉丁小字注音+括号编号「qian (01) zi (02)」(0045,注音不完整,置信度低)。

## 4. 时间语法

- **字卡时长公式:不存在**。回归 `时长 ≈ -0.004×字数 + 1.726`,R²=0.001(n=19)——
  时长由叙事拍点决定:吐槽=8–11 帧、叙述=0.8–2.5s、悬念短句反而最长(「那就是」3 字 3.07s)、
  30 字长卡只给 2.4s(故意读不完)。**生成时按功能查 §1 表,不按字数算**。
- **闪帧**:黑闪 1 帧;抖动闪 2–3 帧;连切爆发每刀 5–6 帧。
- **切换**:100% 硬切。抽检 2 个切点前后帧差(121/79→0)无混合痕迹;46 切点检测尖峰形态一致(抽检样本 n=2)。
- **节奏曲线**:四分段平均镜头长 1.46s → 1.02s(加速)→ 1.58s → 5.55s(骤停收尾);
  模板:**开场 1:1 交替 → 中段加速连发 → 高潮连切爆发 → 收尾骤停长定格**。
- **交替语法**(实测序列):开场严格「字→素→字→素」;中段允许「字字字」连发(0028–0030)
  与「素素素素」连切(0018–0021);全片以「字卡→黑场→人名卡→黑场」收束。

## 5. 反例清单(从统计反推)

- 全片零叠化/零滑动/零缩放过渡 → **禁止任何渐变转场**,只许硬切。
- 全片零无衬线字 → 禁用黑体/无衬线。
- 一卡永远一纯色底 → 禁止渐变底色、禁止一卡多底色。
- 文字卡零图文混排 → 字卡上禁止出现图像;图像镜头上禁止压字幕。
- 零居左/居右对齐的横排正文 → 横排只居中;竖排只右起。
- 字卡时长与字数零相关 → 禁止按"每字 X 秒"生成时长。

## 6. 版权槽位标注(用户自备素材)

- 《物语系列》原片帧:0013, 0015–0022, 0024, 0027, 0031, 0033, 0035, 0038–0041, 0043 → 定义为「动画素材槽位」,复现时由用户自备等价素材,不写入风格规则。
- 实拍库存素材:0001/0003(火焰)、0005(遥控器)、0008(冰饮) → 「实拍空镜槽位」,可自摄/库存替换。
- 图形演出:0010/0037(符号阵列拼成蛇形) → 可用代码生成(非版权素材),归入风格规则。

## 7. 置信度标注(留人工校准)

- 0045 罗马音角注「qian (01) zi (02)」转录不完整(主名四字仅两处注音)。
- 0000 主字下拉丁小字「re」含义不明(疑似注音残留)。
- 0012 顶部黑边内暗红细线,功能不明(样本 n=1)。
- 0024 归类存疑:黑白格栅拼贴可能是图形卡而非原片帧。
- 设问卡/人名卡/骤停黑场各仅 1 例,时长规则样本不足。
- 字体仅能判定"衬线明朝体",具体字型(HG明朝/思源宋/华康)无法从 480p 判定。
- 取色经 16 级量化,±8/通道误差。
